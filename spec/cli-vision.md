# Genshin Wish CLI 目标与范围

## 目标

在本仓库内创建独立 TypeScript CLI 子项目，后续可单独发布为新项目，复用并覆盖现有“非 UI 核心功能”：

- 获取祈愿 URL（自动日志解析 + 手动 URL）
- API 预检、分页抓取、重试、节流
- 多账号本地存储与增量合并
- 导入/导出 UIGF（v4.1 / v3.0）
- 基础抽卡分析输出（CLI JSON/表格）

## 非目标（当前阶段）

- 不实现 Electron UI
- 不实现系统代理抓包（作为后续扩展）
- 不实现 Excel 导出（先保证核心数据链路）

## 成功标准

- 在空目录初始化后，仅依赖 CLI 命令即可完成完整抓取与分析
- 对同一 UID 连续执行抓取不会产生重复记录
- 导出的 UIGF 可被现有工具识别
- 具备清晰的错误码与退出码

# 提瓦特小助手功能研究报告

更新时间：2026-04-14

## 1. 结论概览
提瓦特小助手（YShelper）是面向《原神》玩家的数据分析/辅助工具，核心价值是把“分散在游戏内、需要手算或多次切换界面才能得到的信息”集中成可读的分析结果。最明确、最稳定的功能主轴是：抽卡分析、圣遗物评分、角色练度查询、探索度查询，以及队伍伤害计算。

## 2. 主要功能清单（按可信度分层）

### A. 官方来源明确提到的功能（高可信）

1. 抽卡分析
- 功能做什么：读取祈愿记录并做统计分析。
- 玩家能看到的效果：按卡池查看抽数与结果分布（常驻/角色池/武器池等维度）；直观看到近期与历史抽卡情况。
- 给玩家的便利：不用手动翻游戏历史记录和自己做表，快速判断“当前卡池投入多少、是否继续抽”。

2. 圣遗物评分
- 功能做什么：对圣遗物或角色身上的圣遗物进行打分。
- 玩家能看到的效果：得到可比较的分值/排名结果，便于判断某件圣遗物是否值得继续强化或替换。
- 给玩家的便利：减少“词条好坏全靠感觉”的判断误差，强化决策更快。

3. 角色练度查询
- 功能做什么：对角色养成状态进行综合查询。
- 玩家能看到的效果：能看到角色当前练度相关信息（如构建完成度的综合呈现）。
- 给玩家的便利：判断“是否毕业”“短板在哪”，便于规划树脂和素材投入。

4. 探索度查询
- 功能做什么：查询世界探索相关进度。
- 玩家能看到的效果：区域探索完成情况以百分比或进度化方式呈现。
- 给玩家的便利：快速定位“还差哪些区域没清完”，减少盲图跑图。

5. 队伍伤害查询/计算
- 功能做什么：计算或估算队伍伤害表现。
- 玩家能看到的效果：看到队伍伤害结果（例如对比不同配装/配队方案时的变化）。
- 给玩家的便利：在不反复进本试错的情况下，先做配队筛选。

### B. 社区/分发页常见描述（中可信，建议实测确认）

1. 深渊统计、队伍使用率排行
- 可能实现：按期展示深渊队伍或角色使用情况。
- 预期效果：能看到“本期热门队伍/角色使用率”，用于抄作业或对标环境。
- 便利：新版本开荒期可快速找到主流解法。

2. 角色持有率排行、角色满意度评分
- 可能实现：基于用户数据聚合或投票形成排行。
- 预期效果：看到某角色在样本中的持有占比/主观评分。
- 便利：帮助判断抽取优先级和养成收益。

3. 卡池计时器
- 可能实现：展示卡池剩余时间。
- 预期效果：活动结束倒计时可视化。
- 便利：降低错过池子和活动节点的风险。

4. 抽卡模拟器、圣遗物模拟器
- 可能实现：在工具内进行模拟抽卡/词条强化模拟。
- 预期效果：预览概率体验或强化路线。
- 便利：降低试错焦虑，提高资源规划信心。

## 3. 这些功能在玩家体验上的“具体增益”

1. 从“凭感觉”变成“看数据”
- 典型场景：以前换圣遗物靠主观，现在可直接看评分变化和伤害结果对比。

2. 从“手动抄记录”变成“自动汇总”
- 典型场景：抽卡记录不需要再逐页截屏或手工统计，直接看到分析面板。

3. 从“上线后再试错”变成“离线先决策”
- 典型场景：先在工具里比较配队伤害，再进游戏实战验证，节省体力与时间。

4. 从“碎片化信息”变成“一站式视图”
- 典型场景：抽卡、练度、圣遗物、探索进度可在一个工具链路内完成查询。

## 4. 使用与认知上的注意点

1. 平台形态存在差异
- 官方站点曾描述“微信小程序版本”；而 Google Play / App Store 也存在同名应用条目。不同端功能可能不完全一致。

2. 部分功能可能受权限或数据源限制
- 例如与账号展示设置、数据同步方式有关，某些查询结果可能不是实时毫秒级更新。

3. 免费/会员能力边界需以当前商店页为准
- iOS 页面显示部分功能与订阅相关，实际可用范围要以当前版本说明为准。
